# 简易版剪辑

模仿剪映 Web 功能在一天之内完成一个基本的剪辑功能

## 技术栈

- React
- TypeScript
- Zustand
- ViteJS + Pnpm + Monorepo

## 项目结构

- apps 应用
  - clipper 剪辑应用
- packages
  - store 全局状态
  - types 剪辑数据结构定义
